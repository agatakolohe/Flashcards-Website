Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Variables

Variable naming conventions

Use lowerCamelCase when naming JS variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word. Variables should begin with a letter. They are cAsE sEnSiTiVe (myNumber is a different variable than myNUMBER). Use clear names that describe the value stored. Always name your variables in a manner that will be easy for other developers to understand. Avoid vague letters or initials.

Functions

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

Methods

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.
